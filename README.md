## <img src="resources/images/casaIcone.png" alt="Ícone inicial" width="28"> Sobre o Projeto

O GestOS é uma aplicação desktop em JavaFX voltada para organizar e acompanhar ordens de serviço. O foco atual do projeto é entregar uma interface profissional e visualmente consistente para a gestão de pedidos de trabalho.

## <img src="resources/images/engrenagemConfiguracoesIcone.png" alt="Ícone de configuração" width="28"> Estado Atual

A versão presente neste repositório representa a base inicial do sistema:

- tela inicial criada com FXML
- estilização em CSS
- uso de imagens e ícones localizados em resources/images
- entrada principal na classe Main.java

> Observação: ainda não há implementação de backend, banco de dados, API REST ou integração via Docker neste repositório.

## <img src="resources/images/documentoIcone.png" alt="Ícone de documento" width="28"> Estrutura do Projeto

- Main.java: ponto de entrada da aplicação JavaFX
- view/HomeView.fxml: estrutura da tela inicial
- view/application.css: estilos visuais da interface
- resources/images: arquivos gráficos usados na aplicação

## <img src="resources/images/relogioIcone.png" alt="Ícone de relógio" width="28"> Como Executar

### Requisitos

- JDK 17 ou superior
- IDE com suporte a JavaFX, como IntelliJ IDEA ou VS Code com a extensão Java

### Passos

1. Abra o projeto na IDE.
2. Certifique-se de que as dependências do JavaFX estão configuradas.
3. Execute a classe Main.
